简洁的浏览器扩展，让外文阅读更轻松。支持双语对照、划词翻译、分享卡片。

---

## ✨ 功能

- **🔄 三种模式** - 覆盖 / 双语对照 / 高亮提示
- **⚡ 智能预加载** - 滚动自动翻译，智能缓存
- **🎨 分享卡片** - 选中文字生成精美图片
- **🌍 多语言** - 简中、English、日本语、한국어、Français

---

## 📦 安装

1. 下载本仓库
2. Chrome/Edge → `chrome://extensions` → 开启「开发者模式」
3. 点击「加载已解压的扩展程序」→ 选择项目文件夹

---

## 🚀 使用

| 功能 | 操作 |
|------|------|
| 基础翻译 | 点击扩展图标 → 选语言/模式 → 打开外文网页 |
| 划词翻译 | 选中文字 → 点击「翻译选中」 |
| 分享卡片 | 选中文字 → 点击「生成卡片」→ 下载/复制 |

---

## 🛠️ 技术

- **Manifest V3** - Chrome 扩展标准
- **Intersection Observer** - 性能优化
- **Shadow DOM** - 样式隔离
- **html2canvas** - 图片生成
